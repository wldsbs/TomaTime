# TomaTime
```
간단한 타이머 어플리케이션
- 타이머 설정
- 타이머 완료시 알림
- 무음모드(진동)
- 가로모드 지원
```

## 사용방법
구글플레이스토어에서 <b>토마타임</b> 설치 후 실행해주세요:)  
[토마타임 링크](https://play.google.com/store/apps/details?id=myapplication.app.pomodoro)
